适用场景
类型差异是主要关注点时最优。按总线类型（I2C/HiSport/JTAG）分泳道，类型结构清晰。
优 / 劣
✓ 类型分组结构最清晰
✓ 每条线在泳道内仍可选
✗ 垂直空间占用最多
① 弯折问题：朴素路由 vs 同心弧
② 实际拓扑场景：一对多 · 多方向路由（贴近截图中的布局）
· 悬停高亮连线
弯折核心原则
平行线过弯时，每条线的弧度半径必须递增（同心圆轨道），否则内侧线会叠压。间距 S 固定时，第 i 条线半径 = r₀ + i·S。
推荐策略
✓ 直段用泳道束（省空间）
✓ 转角用同心弧（不重叠）
✓ 一对多分叉用贝塞尔（自然过渡）
