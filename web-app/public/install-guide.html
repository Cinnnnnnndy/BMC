安装部署引导 v0.0.34

BMC Studio CodeX — openUBMC 开发者 VS Code 扩展
环境要求
依赖项最低版本必要性说明
VS Code1.107.0必须扩展运行环境
Node.js18.x本地构建仅手动构建 VSIX 时需要
openUBMC 工具链—推荐生成 MDB 缓存以启用 SR 语言补全
方式一 · VS Code 扩展市场安装（推荐）
1
打开扩展面板
在 VS Code 中按 Ctrl+Shift+X（macOS：⌘+Shift+X）
2
搜索扩展
在搜索框中输入 BMC Studio CodeX 或 openubmcstudio-codex
3
安装并重载
点击「安装」按钮，安装完成后点击「重新加载窗口」即可生效
方式二 · 手动安装 VSIX
1
克隆仓库并安装依赖
git clone <repo-url> bmcstudio_codex cd bmcstudio_codex npm install
2
构建 VSIX 包
npm run plugin-package # 生成：openubmcstudio-codex-0.0.34.vsix
3
安装到 VS Code
code --install-extension openubmcstudio-codex-*.vsix
或在扩展面板点击「···」→「从 VSIX 安装」
配置 MDB 缓存（SR 语言服务器）
SR 语言服务器从以下路径读取 openUBMC MDB 类定义，用于 .sr 文件自动补全。
~/.cache/openubmc/mdb_cache/
💡
使用 openUBMC 工具链 构建项目后，运行 ubmc export-mdb 将 MDB 导出至该目录。 导出完成后 SR 语言服务器会在下次启动时自动加载新定义。
功能速览
🔍JSON 北向接口辅助
定义跳转、悬停提示、自动补全、实时错误检测，适用于北向 API .json 配置文件
📄SR 文件预览
类似 Markdown 的实时预览，自动检测并合并多个 .sr 文件（主文件 + 扩展构型）
🧮管道表达式计算
CodeLens 触发，支持调试模式（实时计算）和用例模式（批量测试验证 + 导出）
⚙️SMC 偏移量计算
可视化分解 32 位 SMC 命令字，实时水印解码，支持多进制输入格式
🌐SR 语言服务器
基于 LSP 的 .sr 文件补全、诊断与悬停提示，实时扫描 MDB 类定义
📋MIB 文件支持
ASN.1 MIB 文件语法高亮与自定义图标，MIB 对象重复配置校验
